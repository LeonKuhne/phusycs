CONTROLS

click - add new particles
right click - create parent and connect selected
click while selected - place particle in orbit around target
click particle - select
shift - slowmo, speed/scale inputs are amplified
shift + drag/click particle - select additional particles
drag - select in range
drag particle - move if static
ctrl + drag - pan world around
scroll - scale all orbits
scroll selected - accelerate particles orbit
scroll selected while paused - scale particles orbit
scroll selected root - scale orbits in system
space - pause
esc - deselect
delete - delete selected
'a' - select all
'c', 'd' - connect/disconnect edges between selected particles
'm', 's' - mute/solo selected edge
'r' - adjust selected speeds randomly
'=+-/*^' - open math menu

MATH MENU

'#' reference orbit depth
'r' assign unique random value (0-1) for selected
